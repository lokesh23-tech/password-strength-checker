# password-strength-checker
"A Python tool that checks password strength"
Title: Password Strength Checker

What it does: A Python script that evaluates password strength based on length, uppercase/lowercase letters, numbers, and special characters, and gives suggestions to improve weak passwords.

Why I built it: To understand common password vulnerabilities and how weak passwords can be exploited.

How to run: Open in Google Colab or any Python environment, run the script, and enter a password to test.
